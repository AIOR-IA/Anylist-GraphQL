# Dev

1. Clone the project
2. Copy the ```env.template``` and rename to ```.env```
3. Run
```
npm install
```
4. Up the image (Docker desktop)
```
docker-compose up -d
```

5. Up de backend on NestJS
```
npm start:dev
```

6. Open the site
```
localhost:3000/graphql
```
